# datakx
一套免费全场景开源电商系统Java版
一整套供应链平台+SAAS商城+自营商城、多商户商城、积分商城、跨境商城系统，支持S2B2C、B2B、B2C、O2O等全场景模式，开源稳定支持二次开发。 商城支持PC端、APP、小程序、H5满足各种场景需求。
